Documentation

This project would not have been doable without the fantastic libraries and code sources made by others. We would like to properly acknoledge those sources and allow other coders to utilize these great libraries too.

  • Leaflet - A javascript library to add maps to your webpage
  • OpenStreetMap (OSM) - The greatest provider of maps out there. While we used Cloudmade, at least initially, for our tile source, OSM just has a lot more features and is more open and customizeable.
  • Bootstrap - You would be admiring our site right now if it wasn't for this click library. A must-have for rapid frontend design.
  • MTA - We couldn't have gotten our transport updates without the people who created and monitor the feeds.
  • JQuery - It goes without saying that our JavaScript coding was made a lot easier thanks to JQuery.

Alternative resources (these guys nearly made the cut!)
  • OpenLayers - It took a while to decide upon Leaflet because OpenLayers seemed like such a great option. Definitely will be using this library in the future!
  • Mapstraction - We only have one map, but this was another great option we could have used.
